# Alarm Bot

# How To Install 

```sh
git clone https://github.com/PlusTM/Alarm

cd Alarm

chmod +x start.sh

./start.sh install

chmod +x tg

```
set Sudo user Id in line 8 Alarm.lua and run:
```sh
./start.sh
```
For Send Online Status Message Go To A chat And Send **/setrealm** Command And Relunch Again
